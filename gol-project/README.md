Conway's Game of Life in Erlang
=============
By Jonatan and Anders

Instructions
-------------
This will generate a glider on a 40x40 frame.

* Compile files
	* c("ehtml.erl").
	* c("frame.erl").
	* c("gol.erl").

* gol:gol(40,40,[{5,4},{3,5},{5,5},{4,6},{5,6}]).

* Point your browser to http://localhost:8088/
